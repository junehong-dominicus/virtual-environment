| Copy Line Below | Shift+Alt+(downarrow) | Option+Shift+(downarrow) |
| Multiline-select Cursor | Ctrl+Alt+(up/downarrow) | Cmd+Option+(up/downarrow) |
| Multi-select Cursor | Ctrl+D | Cmd+D |
| Copy Line | Ctrl+Alt+D | Cmd+Shift+D |
| Duplicate Line | Ctrl+Shift+L | Cmd+Shift+L |
